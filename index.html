What is html?

HTML It is the language used to set up web pages. Html consists of elements which are the building blocks of web development, these consist of content and tags.

Html is a language that allows you to add tags and images to plain text to give meaning and structure to the webpage.

This includes hyperlinks which allow you to jump from one piece of content to another using an anchor element and clicking on them. ie. Website>Visit W3Schools.com!

HTML stands for
  • Hyper
  • Text
  • Markup
  • Language

Markup Language add tags to plain text to define meaning. HTML controls the structure of the webpage to include content and words.

What is CSS?

CSS controls the presentation of the webpage to include layout, colour of text and background, size of the html elements and fonts.

Essentially CSS defines the rules for how the HTML will be displayed. The style or beauty of the webpage.

Together these make a CSS rule:

1. Selectors
2. Properties
3. Values

You can add CSS to HTML in 3 ways

  1. Inline styles
  2. Internal CSS
  3. External Spread sheet
CSS stands for CASCADING STYLE SHEETS

Pairing these wih HTML makes a web design.
